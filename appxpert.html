AppXpert Installation Instructions

  1. Go to your Device Settings -> Security and Enable "Allow installation of application from both trusted and unknown sources" (This will enable you to install apps from sources other than Play Store)
  2. Download and Install the 'App Xpert'
  3. Longpress on homescreen and select 'Widgets' from the list
  4. Select and drag 'App Xpert' widget to home screen on your device.
